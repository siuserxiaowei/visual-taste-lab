# Visual Taste Lab

Visual Taste Lab is a UI/UX design-language skill for Codex and Claude-style coding agents.

It helps an agent avoid generic AI-looking webpages by doing a VI audit before writing page code:

- Logo or wordmark anchor
- Brand primary, secondary, accent, neutral, and semantic colors
- Site type: company site, product page, transactional utility, content brand, dashboard, portfolio
- Typography, spacing, cards, buttons, tables, CTAs, imagery, and anti-patterns

## Links

- Skill entry: [`visual-taste-lab/SKILL.md`](visual-taste-lab/SKILL.md)
- Public preview: GitHub Pages site for this repository

## Example Command

```text
/goal Use $visual-taste-lab to improve this project visually.

Do not redesign immediately. Start with a VI audit:
- Identify the logo or wordmark.
- Identify or propose primary color, secondary color, accent color, neutral palette, and semantic colors.
- Decide whether this is a company website, product website, transactional utility, institutional page, dashboard, portfolio, or another type.
- Explain how the visual identity supports the audience.

Then apply the design language to the key pages/components.
```
